    HTML Elements Pre Lesson

    Overview

    1. In your own words, define an HTML element.
      2. An HTML element is a section of a webpage.
    2. What is a void element?
      3. A void element has no closing tags.
    3. What do all elements begin and end with?
      4. All elements begin and in with opening and closing tags.
    4. What is the syntax of an HTML element?
      5. The name of the element goes inside the opening and closing tag to tell the html document what type of element it is and how it wants the page to display.
    5. What do we call an element that contains other elements?
      6. An element that contains other elements is a block-level element.
    6. What are the nested elements called?
      7. The nested elements are called inline elements.

    Block-Level vs. Inline Elements

    1. What is a block level element? What is an example of a block level element?
      2. A block-level element occupies the entire space of itsparent element.
    2. What is an inline element? What is an example of an inline element?
      3. An inline element only occupies the space between the tags.

    Attributes

    1. What is an attribute?
      2. Attributes describe elements.
